# 环保小助手多模式切换实现方案

## 目标
在环保小助手页面顶部增加切换按钮，支持选择 **Java LangChain4j** 或 **Python LangChain** 两种模式。

## 方案选择
**推荐：Java转发方案**（前端 → Java → Python）

| 方案 | 优点 | 缺点 |
|------|------|------|
| Java转发 | 认证统一、单一入口、便于管理 | 多一层转发 |
| 前端直调 | 延迟低 | 认证不统一、CORS问题 |

## 二、后端实现（待实现）

### 步骤1：修改 AssistantController.java

**文件位置**：`back-end/src/main/java/com/environment/controller/AssistantController.java`

添加以下内容：

```java
// 1. 新增依赖注入
@Value("${python.api.url:http://localhost:5001}")
private String pythonApiUrl;

private WebClient webClient;

@Autowired
public void setWebClient(WebClient.Builder webClientBuilder) {
    this.webClient = webClientBuilder.baseUrl(pythonApiUrl).build();
}

// 2. 新增接口方法
@PostMapping(value = "/chat-python", produces = "text/html;charset=utf-8")
public Flux<String> chatPython(@RequestBody ChatRequest request) {
    return webClient.post()
            .uri("/api/chat")
            .bodyValue(new PythonChatRequest(request.getMessage()))
            .retrieve()
            .bodyToFlux(String.class)
            .map(response -> {
                // 解析Python API返回的JSON，提取answer字段
                // Python返回格式: {"question":"...", "answer":"...", "sources":[...]}
                try {
                    ObjectMapper mapper = new ObjectMapper();
                    JsonNode node = mapper.readTree(response);
                    return node.path("answer").asText();
                } catch (Exception e) {
                    return response;
                }
            });
}

// 3. 新增内部DTO类
public static class PythonChatRequest {
    private String question;

    public PythonChatRequest(String question) {
        this.question = question;
    }

    // getter/setter
}
```

### 步骤2：添加必要依赖（如果缺失）

在 `pom.xml` 中确认已有：
```xml
<!-- WebFlux（已有） -->
<dependency>
    <groupId>org.springframework.boot</groupId>
```

如需JSON解析，添加：
