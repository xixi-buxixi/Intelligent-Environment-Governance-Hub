# 中国生态环境保护政策法规概览

## 一、法规标准体系

### 主要法律法规
- 《中华人民共和国环境保护法》（2014年修订）
- 《中华人民共和国环境影响评价法》（2003年9月1日施行）
- 《中华人民共和国固体废物污染环境防治法》
- 《城市市容和环境卫生管理条例》

### 标准体系
- 《环境空气质量标准》
- 《土壤环境监测技术规范》
- 《饮用水水源保护区勘界技术指南》
- 《入河入海排污口监督管理技术指南》
- 《生态工业园区建设标准》

## 二、主要业务领域

### 中央生态环境保护督察
中央生态环境保护督察是党中央、国务院推进生态文明建设和生态环境保护的重大制度安排。督察内容包括：
- 贯彻落实党中央、国务院生态环境保护决策部署情况
- 地方党委政府生态环境保护责任落实情况
- 突出生态环境问题及其处理情况

### 环境影响评价管理
建设项目环境影响评价实行分类管理：
- 编制环境影响报告书
- 编制环境影响报告表
- 填报环境影响登记表

### 排污许可管理
《全面实行排污许可制实施方案》明确了：
- 排污许可分类管理名录
- 排污许可证申请与审批
- 排污许可证后监管

### 应对气候变化
主要工作包括：
- 全国碳排放权交易市场管理
- 温室气体清单编制
- 碳达峰碳中和目标落实
- 国家温室气体排放因子数据库建设
