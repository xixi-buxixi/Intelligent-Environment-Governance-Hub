# 环境保护基础知识

## 什么是环境保护？

环境保护是指人类为解决现实的或潜在的环境问题，协调人类与环境的关系，保障经济社会的持续发展而采取的各种行动的总称。

## 主要环境问题

### 1. 大气污染

大气污染是指大气中污染物的浓度达到有害程度，破坏生态系统和人类正常生存发展的条件，对人和生物造成危害的现象。

**主要来源**：
- 工业废气排放
- 汽车尾气
- 燃煤取暖
- 建筑扬尘

**防治措施**：
- 发展清洁能源
- 提高燃油品质
- 植树造林
- 推广公共交通

### 2. 水污染

水污染是指水体因某种物质的介入，导致其化学、物理、生物特性的改变，影响水的有效利用，危害人体健康或破坏生态环境。

**主要来源**：
- 工业废水
- 农业面源污染
- 生活污水
- 畜禽养殖废水

**防治措施**：
- 建设污水处理厂
- 推广清洁生产
- 控制化肥农药使用
- 保护水源地

### 3. 土壤污染

土壤污染是指因人为因素导致某种物质进入土壤，引起土壤化学、物理、生物特性的改变，影响土壤功能和有效利用的现象。

**主要来源**：
- 工业废渣
- 农药化肥
- 大气沉降
- 污水灌溉

**防治措施**：
- 控制污染源
- 土壤修复
- 农业标准化生产
- 加强监测监管

### 4. 固体废物污染

固体废物污染是指在生产建设、日常生活和其他活动中产生的污染环境的固态、半固态废弃物质。

**分类**：
- 工业固废
- 生活垃圾
- 危险废物
- 农业废弃物

**处理方式**：
- 减量化：减少产生量
- 资源化：回收利用
- 无害化：安全处置

## 环境保护法律法规

### 《中华人民共和国环境保护法》

2015年1月1日起施行，是我国环境保护领域的基础性法律。

**基本原则**：
- 保护优先
- 预防为主
- 综合治理
- 公众参与
- 损害担责

### 其他重要法律

- 《大气污染防治法》
- 《水污染防治法》
- 《土壤污染防治法》
- 《固体废物污染环境防治法》
- 《环境影响评价法》

## 碳达峰与碳中和

### 什么是碳达峰？

碳达峰是指二氧化碳排放量达到历史最高值，之后经历平台期进入持续下降的过程。

### 什么是碳中和？

碳中和是指企业、团体或个人在一定时间内直接或间接产生的温室气体排放总量，通过植树造林、节能减排等形式，以抵消自身产生的二氧化碳排放量，实现二氧化碳"零排放"。

### 中国的双碳目标

- **碳达峰**：2030年前实现二氧化碳排放达峰
- **碳中和**：2060年前实现碳中和

## 个人如何参与环境保护？

1. **减少能源消耗**
   - 随手关灯
   - 使用节能电器
   - 合理使用空调

2. **绿色出行**
   - 选择公共交通
   - 骑行或步行
   - 拼车出行

3. **减少废弃物**
   - 减少一次性用品使用
   - 自带购物袋
   - 做好垃圾分类

4. **节约水资源**
   - 一水多用
   - 及时修复漏水
   - 使用节水器具

5. **保护生态环境**
   - 不乱扔垃圾
   - 不捕杀野生动物
   - 参与植树造林

## 环境监测指标

### 空气质量指数（AQI）

| AQI数值 | 空气质量级别 | 对健康影响 |
|---------|-------------|-----------|
| 0-50 | 优 | 空气质量令人满意 |
| 51-100 | 良 | 空气质量可接受 |
| 101-150 | 轻度污染 | 敏感人群应减少户外活动 |
| 151-200 | 中度污染 | 应减少户外活动 |
| 201-300 | 重度污染 | 应避免户外活动 |
| >300 | 严重污染 | 应避免一切户外活动 |

### 水质分类标准

| 类别 | 适用范围 |
|------|---------|
| I类 | 源头水、国家自然保护区 |
| II类 | 集中式饮用水源地一级保护区 |
| III类 | 集中式饮用水源地二级保护区 |
| IV类 | 一般工业用水区 |
| V类 | 农业用水区 |
