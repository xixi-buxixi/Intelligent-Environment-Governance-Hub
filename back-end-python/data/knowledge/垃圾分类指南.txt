# 垃圾分类指南

## 什么是垃圾分类？

垃圾分类是指按照垃圾的不同成分、属性、利用价值以及对环境的影响，并根据不同处置方式的要求，分成属性不同的若干种类。

## 垃圾分类的四大类别

### 一、可回收物（蓝色垃圾桶）

可回收物是指适宜回收和资源利用的垃圾，包括：

1. **纸类**：报纸、书本、纸箱、信封等
2. **塑料**：塑料瓶、塑料桶、塑料玩具等
3. **金属**：易拉罐、金属罐头盒、旧锅具等
4. **玻璃**：玻璃瓶、玻璃杯、镜子等
5. **织物**：旧衣服、床单、毛巾等

**投放要求**：
- 轻投轻放
- 清洁干燥，避免污染
- 立体包装应清空压扁后投放
- 易碎物品应包裹后投放

### 二、有害垃圾（红色垃圾桶）

有害垃圾是指对人体健康或自然环境造成危害的垃圾，包括：

1. **电池类**：废电池、蓄电池等
2. **灯管类**：荧光灯管、节能灯等
3. **药品类**：过期药品、药品包装等
4. **化学品**：油漆、杀虫剂、消毒剂等
5. **电子产品**：废旧手机、电路板等

**投放要求**：
- 应轻投轻放
- 易破损物品应连带包装或包裹后投放
- 如易挥发物品应密封后投放

### 三、厨余垃圾（绿色垃圾桶）

厨余垃圾是指居民日常生活及食品加工、饮食服务等活动中产生的垃圾，包括：

1. **食材废料**：菜叶、果皮、蛋壳、鱼骨等
2. **剩饭剩菜**：米饭、面条、肉类残渣等
3. **过期食品**：过期零食、变质食品等
4. **植物类**：花卉、落叶等

**投放要求**：
- 纯流质食物垃圾应直接倒进下水口
- 有包装物的应去除包装后投放
- 避免混入餐具、牙签等杂质

### 四、其他垃圾（灰色垃圾桶）

其他垃圾是指除可回收物、有害垃圾、厨余垃圾以外的垃圾，包括：

1. **卫生纸**：餐巾纸、卫生纸、尿不湿等
2. **陶瓷制品**：破碎陶瓷、花盆等
3. **一次性用品**：一次性餐具、烟蒂等
4. **灰土**：烟灰、尘土等

## 垃圾分类的意义

1. **减少环境污染**：有害垃圾得到专门处理，减少土壤、水源污染
2. **资源回收利用**：可回收物变废为宝，节约资源
3. **减少填埋压力**：厨余垃圾可堆肥处理，减少填埋量
4. **提升环保意识**：培养公民环保习惯，建设美丽家园

## 常见问题解答

**Q：用过的纸巾是什么垃圾？**
A：用过的纸巾属于其他垃圾，因为使用后被污染，无法回收利用。

**Q：大骨头是什么垃圾？**
A：大骨头属于其他垃圾，因为它难以腐烂且硬度较高，不适合厨余垃圾处理设备。

**Q：过期化妆品是什么垃圾？**
A：过期化妆品属于有害垃圾，因为其中含有对人体和环境有害的化学物质。
