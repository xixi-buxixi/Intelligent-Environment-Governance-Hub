# 垃圾分类

## 定义
垃圾分类（Garbage classification），一般是指按一定规定或标准将垃圾分类投放、收集、运输和处理，从而转变成公共资源的一系列活动的总称。

## 目的
垃圾分类的目的是提高垃圾的资源价值和经济价值，减少垃圾处理量和处理设备的使用，降低处理成本，减少土地资源的消耗，具有社会、经济、生态等几方面的效益。

## 分类过程
垃圾在分类储存阶段属于公众的私有品，垃圾经公众分类投放后成为公众所在小区或社区的区域性公共资源，垃圾分类搬运到垃圾集中点或转运站后成为没有排除性的公共资源。从国内外各城市对生活垃圾分类的方法来看，大多都是根据垃圾的成分、产生量，结合本地垃圾的资源利用和处理方式等来进行分类的。

## 垃圾分类标准

### 厨余垃圾
厨余垃圾是指居民日常生活及食品加工、饮食服务、单位供餐等活动中产生的垃圾，包括丢弃不用的菜叶、剩菜、剩饭、果皮、蛋壳、茶渣、骨头（鸡骨、鱼刺类）等，其主要来源为家庭厨房、餐厅、饭店、食堂、市场及其他与食品加工有关的行业。

### 有害垃圾
有害垃圾是指生活垃圾中对人体健康或自然环境造成直接或潜在危害的废弃物，需单独收集、运输、存贮并由环保部门认可的专业机构进行特殊安全处理。

**类型包括：**
- 废电池（镉镍电池、氧化汞电池等）
- 废荧光灯管
- 过期药品
- 废弃化妆品
- 废油漆及其包装物
- 含水银温度计
- 杀虫剂容器等

**投放要求：**
- 轻放易损物品并采取密封防护
- 压力罐需排空内容物
- 破碎物品需包裹后投放
- 公共场所产生的有害垃圾若无专用收集容器，需携带至指定投放点，禁止混入其他垃圾桶

### 其他垃圾
其他垃圾，指除可回收物、有害垃圾、厨余垃圾外的其他生活垃圾，以及难以辨识类别的生活垃圾。

**主要包括：**
- 受污染与无法再生的纸张
- 一次性餐具
- 破旧陶瓷品
- 烟头
- 灰土等难以回收的废弃物

其他垃圾一般采用焚烧发电或卫生填埋等方式进行无害化处理。

### 可回收物
可回收物指生活垃圾中未经污染、适宜回收循环利用的废物，属于再生资源，可通过再加工成为生产原料或再利用物品。

**主要类别包括：**
- 废纸
- 塑料
- 玻璃
- 金属
- 纺织物
- 电子废弃物等

## 中国垃圾分类发展历程

### 古代
- **唐朝时期**：长安城作为当时世界上最大的城市，《唐律疏议》规定随地扔垃圾要打六十打板，还有专门以清理垃圾、粪便为职业的人
- **宋朝时期**：大街小巷已经出现专职的环卫工人，官府设置了"街道司"专门机构
- **明清时期**：城市和农村都已经形成回收的完备产业链

### 近现代
- **1957年**：北京市提出了"城区分类收集垃圾"的构想，《北京日报》头版头条刊登《垃圾要分类收集》
- **1989年**：全国人大常委会颁布《中华人民共和国环境保护法》
- **1992年**：国务院颁布《城市市容和环境卫生管理条例》，规定要对生活垃圾逐步进行分类收集
- **1996年**：实施《固体废物污染防治法》，北京西城区大乘巷居民开始垃圾分类实践
- **2000年**：原建设部将北京、上海、广州、深圳、杭州、南京、厦门、桂林八个城市作为生活垃圾分类收集试点城市
- **2003年**：国家出台《城市生活垃圾分类标志》，将生活垃圾划分为可回收物、有害垃圾和其他垃圾三类

## 现行分类标准
目前，生活垃圾普遍按照可回收物、有害垃圾、厨余垃圾、其他垃圾进行"四分类"。随着垃圾处理技术的发展，处理方式从过去以填埋为主，逐步转向以现代化焚烧发电为核心的资源化利用模式。
